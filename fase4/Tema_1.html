FLEXBOX

El módulo de caja flexible conocido como Flexbox, es un módulo de diseño creado para CSS3 , fue
diseñado como modelo unidimensional de Layout que ayuda a distribuir mejor los espacios entre
ítems y mejorar su alineación.

El módulo de flexbox se trabaja pensando en dos ejes, el eje principal y el eje cruzado. El eje
principal se define como ‘Flex-direction’ y puede poseer cuatro valores: row, row-reverse, column
y column reverse; si elige row o row-reverse trabajará a lo largo de las filas, si elije column o
column-reverse trabajará desde el borde superior hasta el final.

EL área del documento que tiene un flexbox es llamado contenedor flex, Para crearlo se establece
la propiedad ‘Display’ como flex o inline-flex, una vez hecho esto los elementos contenidos en él
se vuelven ‘ítems flex’

Algunas de las propiedades del Flexbox son:

Display: Flex

Al utilizar las propiedades del Flexbox es necesario crear
un contenedor master para que configure las características de
otros contenderes junior, al contendeor master se declara la propiedad

por ejemplo:

#Container {
display: flex;
}

Flex-wrap

Con la propiedad 'Flex-wrap' es posible indicar cuales elementos
se deben trasladar cuando el contenedor no tiene el espacio suficiente

los valores que se pueden adaptar del Flex-wrap son:

-nowrap: Los elementos no se pasan a la fila siguiente y se reduce el ancho
para poder mostrarlos

-wrap: los elementos pasan a la fila siguiente y no semodifica el ancho

-wrap-reverse: los elementos pasan a la siguiente fila, pero al contrario
de como fueron declarados

por ejemplo:

#Container {
display: flex;
flex-wrap: wrap;
}

Justify-content

Con la propiedad Justify-content es posible definir la justificación
horizontal de los elementos junior de un contenedor flexible

los valores que se pueden adaptar del Justify-content son:

-flex-start: posiciona los elementos a la izquierda del contendor

-flex-end: posiciona los elementos a la derecha del contendor

-center: posiciona los elementos al centro del contendor

-space-between: añade un espacio idéntico entre los elementos
Estando el primero a la izquierda y el ùltimo a la derecha

-space-around: espacia de forma regular los elementos que no están
alineados a la izquierda o la derecha del contenedor.

por ejemplo:

#Container {
display: flex;
justify-content: flex-start;
}
