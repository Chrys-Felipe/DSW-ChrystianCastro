CSS GRID

El CSS contiene funciones de diseño que van encaminadas para todos los desarrolladores
de contenido páginas WEB, se destaca principalmente porque permite dividir la página
en varias áreas o regiones, por definir la relación en términos de tamaño, posición y
capas entre partes de un control construido a partir de HTML.

De igual manera que en las tablas, el grid-layout permite alinear en columnas y filas
los elementos, no obstante, con CSS grid hay mayor variedad de diseños y de manera
más sencilla que en las tablas.

El CSS Grid se puede utilizar de la siguiente manera:

Creación de contenedores


Cuando se trabaja con CSS Grid se utilizan dos unidades independientes:Contenedores y elementos

El contenedor se encuentran en un nivel superior de los elementos y en él se definen
las propiedades que están dentro de un contenedor.

Rejillas, Columnas y Filas:


Cuando se trabaja con CSS Grid se trabaja con filas y columnas que crean cuadrículas

y en ellas se ubican y distribuyen los distintos elementos, y es el usuario quien
define el tamaño de las filas y las columnas añadiendo las propiedades al contendor.
la opción 'grid-gap' genera un espacio vacío.

Dispocición de los elementos:


Una vez que ya se han definido las filas y columnas, en ellas ubicamos los elementos,
Como primer paso se ben crear los elementos y especificar los valores de inicio y fin
sin embargo un elemento no debe ocupar necesariamente una sola celda dentro de la cuadricula.
